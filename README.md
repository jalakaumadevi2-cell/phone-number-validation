# Phone Number Validation System (+91 Support)

## 📌 Description
This project validates Indian phone numbers with or without country code (+91).

## ✅ Features
- Validates 10-digit numbers
- Supports +91 country code
- Checks numeric format
- Ensures valid starting digits (6-9)

## 🧠 Concepts Used
- String handling
- Conditional statements
- Functions

## ▶️ How to Run
1. Install Python
2. Run the file:
   python phone_validation.py

## 📊 Example
Input: +919876543210  
Output: Valid
